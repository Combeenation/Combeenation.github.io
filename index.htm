Getting Started with Hive

Hive is the scripting language used to build Combeenation configurators.
Here you can learn more about how to write scripts and rules to build your own configurator.

Info

This documentation is still a work in progress!
We are actively working on completing it, so check back frequently for updates!

You can explore the topics on the left to learn more about the various features of the language, or read on for some general information.

Comments

Comments allow you to write down helpful information directly in your scripts. They have no effect whatsoever on the script itself, but they can help you or others better understand what the script is all about.

To write a comment, start with two slashes // and then write whatever you want. Note that your comment takes up the remainder of the line.
Let's see an example:

Amount * 3  // Hi! I'm a comment.

Here we see a very basic script that multiplies the Amount component with 3 and a not very helpful comment next to it.
