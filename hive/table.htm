The RecordComponent

Functions

Name Description
get Returns the record with the specified key
getFirst Returns the first record of a table
Info

The examples below assume that there is a RecordComponent called fruits that looks like the one below.

Key Name Price
1 "Apples" 10
2 "Oranges" 20
3 "Bananas" 30

If you want to use the table in your own scripts to try out the examples, this is how it looks like in Hive:

{
  { Key=Number, Name=Text, Price=Number }
  {          1,  "Apples", 10           }
  {          2, "Oranges", 20           }
  {          3, "Bananas", 30           }
}

Get

Returns the record with the specified key, or Empty if none of the records has that key.

Example Result
fruits.Records.get(1) [ 1,"Apples", 10]
fruits.Records.get(3) [ 3,"Bananas",30]
fruits.Records.get(9) Empty
Info

Tip: use the getFirst() function to provide a fallback record if none was found by Get:

fruits.Records.get(9) or fruits.Records.getFirst()

GetFirst

Returns the first record of a table, or Empty if there are no records.

Example Result
fruits.Records.getFirst() [ 1,"Apples", 10]
