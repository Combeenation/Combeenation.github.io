Result Visualization

The result panel automatically converts the output data in a human-readable format.

Record data

  • Table which renders columns based on it's type
  • Easy navigation
  • Automatic paging for big data
  • Several expand/collapse features

Column rendering

  • Nested data: (Expandable) inner table
  • Logic: Displayed as checked/unchecked box
  • Image: Directly loads the image url
    • If the overall size is too large for efficient loading, it will show only an icon
    • Hover over the image for a larger preview

Pan / Navigate

The right mouse button can be used to pan inside larger tables. Holding the right mouse button will hide the cursor and you can move in any direction until it's released.

Minimize / Maximize columns

Every column can be minimized by moving over it's header and clicking the - icon.

Collapse all columns except the clicked one

It's possible to focus on one column by clicking on the header with any modifier key held down (ctrl, alt, shift).

To reverse the action and expand all columns again, click on the focused column again with any modifier key.

Expand / Collapse nested data

When a component contains nested data (e.g. record inside record-column) it's collapsed by default. The nested data can be expanded individually per row and nesting.

JSON

Plain text which is parseable as JSON gets automatically formatted.

Buttons (bottom toolbar)

  • Toggle raw data: In case the original data is needed (e.g. for using the result as component input elsewhere)
  • Expand / Collapse all: Expand or collapse all inner tables recursively
  • Copy result: Copies the whole content into the clipboard
    • Only available when raw data is displayed
