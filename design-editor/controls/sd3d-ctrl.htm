ShapeDiver3d control

Controlling the camera

You can position the camera using the controls Set camera position slot. This slot expects 2 parameters:

  • Pos: The target position
  • TransitionMS: Duration for the camera movement from the current position to the given target position in [ms]. 0 for no transition.

The Pos parameter expects a string in the following format:

{
    "position": {
        "x": 0,
        "y": 0,
        "z": 0
    },
    "target": {
        "x": 0,
        "y": 0,
        "z": 0
    }
}

Centering camera

The camera can be centered using the following components & rules. The component names are only exemplary and can be chosen to your likings. It is recommended to place them all into a common group like Center 3d camera.

// Component "ModelHalfWidth"
{ModelWidth} / 2 // TODO: Insert appropriate values for the placeholder {ModelWidth}

// Component "ModelHalfHeight"
{ModelHeight} / 2 // TODO: Insert appropriate values for the placeholder {ModelHeight}

// Component "CameraFieldOfView"
37 // degree

// Component "CameraDistanceWidth"
ModelHalfHeight / Math.tan(CameraFieldOfView / 2)

// Component "CameraDistanceHeight"
ModelHalfWidth / Math.tan(CameraFieldOfView / 2)

// Component "CameraDistance"
Math.max(CameraDistanceHeight, CameraDistanceWidth) * -1

// Component "CenterCameraPos"
"""{{
    "position": {{
        "x": {=ModelHalfWidth},
        "y": {=CameraDistance},
        "z": {=ModelHalfHeight}
    },
    "target": {{
        "x": {=ModelHalfWidth},
        "y": 0,
        "z": {=ModelHalfHeight}
    }
}"""

Finally you only have to connect the component CenterCameraPos to the Set camera position slot of your ShapeDiver3d control.
