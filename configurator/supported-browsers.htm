Supported browsers

The following browsers are officially supported by Combeenation:

  • Chrome 60+
  • Firefox 57+
  • Safari 10+
  • Edge 15+

Changelog

Mar 2019

Before this date, we supported the following browsers:

  • Chrome 60+
  • Firefox 57+
  • IE 11+
  • Safari 10+
  • Edge 15+

Version 1.21.5

Before this release, we supported the following browsers:

  • Android platform (mobiles, tablets etc.):
    • Firefox 40+
    • WebKit 536+ (~ Chrome 19)
  • All other platforms (Desktop PC, iPhone, iPad, ...):
    • Firefox 24+
    • IE 10+
    • Safari 6.1+
    • WebKit 536+ (~ Chrome 19)
